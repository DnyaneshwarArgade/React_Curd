# React_Curd

This is a simple CRUD (Create, Read, Update, Delete) application built using **React + Vite**.  
The project demonstrates basic form handling, state management, and UI rendering.

---

## Setup Instructions

Follow the steps below to run the project locally:

```bash
# Clone the repository
git clone https://github.com/<your-username>/React_Curd.git

# Go to project folder
cd React_Curd

# Install dependencies
npm install

# Start development server
npm run dev
